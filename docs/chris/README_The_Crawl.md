---

## Overview

*Return of the Exiled: Where’s My Sandwitch* is a top-down, skill-tree-based action RPG built in Unity. The player controls an exiled hero fighting through levels of enemies to grow stronger, earn loot, and eventually face the final boss—all while uncovering the mystery of their missing sandwich.

This README provides simple steps to **run the executable** on any Windows PC with minimal setup.

---

## System Requirements

**Minimum:**  
- **OS:** Windows 10 or newer (64-bit)  
- **Processor:** Intel i5 / AMD equivalent  
- **Memory:** 8 GB RAM  
- **Graphics:** DirectX 11 compatible GPU  
- **Storage:** ~1 GB free space
